Welcome to parabam
******************

:Author: Henry Farmery
:Date: |today|
:Version: |version|

`parabam` is a python package that allows users to quickly and easily conduct analysis on BAM files. `parabam` is designed to allow the user to make maximum useage of the processing power available to thier machines.

Running an analysis on a BAM file can be as easy as specifiying a single function and then typing a command into on your terminal. More advanced users may incorporate `parabam` into thier own code or expand `parabam` by making using of OOP inheritence.
